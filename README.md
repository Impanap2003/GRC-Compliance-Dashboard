GRC Compliance Dashboard & Continuous Monitoring System

Overview
This project is a web-based GRC dashboard developed to monitor risks, compliance, and security incidents.

Features
- Risk tracking (Open/Closed)
- Compliance monitoring
- Risk prioritization
- Interactive charts (Pie & Bar)
- Real-time data simulation using JSON
- Dark mode UI

Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

How to Run
1. Open project in VS Code
2. Run using Live Server

Project Structure
- index.html → Dashboard UI
- data.json → Dynamic data

Future Improvements
- Backend integration (Flask/Node.js)
- Database connectivity
- User authentication
